###  $1\degree$Generic equation of the type: $f(x) = k, k = constant$

$sin (x) = k$
$x_1 = a + 2\pi n, n \in \mathbb{z}$
$x_2 = (\pi - a) + 2\pi n, n \in \mathbb{z}$ 

$cos(x) = k$ 
$x_1 = a + 2\pi n, n \in \mathbb{z}$
$x_2 = -a + 2\pi n, n \in \mathbb{z}$

$tan(x) = k$
$x = a + \pi n, n \in \mathbb{z}$
